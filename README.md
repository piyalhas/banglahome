# বাংলা হোমস - Bangladeshi House Rental Platform

A full-stack web application for renting properties in Bangladesh.

## Features
- User Registration & Authentication
- Property Listings with Search & Filters
- Property Management Dashboard
- Contact Forms
- Responsive Design
- Bengali Language Support

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT
